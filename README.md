### 1. **Darc** - Real-time Communication Platform
**Repository**: [https://github.com/bhargavmane1802/Darc](https://github.com/bhargavmane1802/Darc)

A modern real-time communication platform built with cutting-edge web technologies.

**Tech Stack**:
- **Frontend**: React 19 with Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Real-time**: Socket.io
- **AI Integration**: Google Generative AI

- **Security**: Helmet, CORS

**Key Features**:
- Real-time messaging with Socket.io
- User authentication and authorization
- AI-powered features using Google's Generative AI
- File uploads with Cloudinary integration
- Scheduled tasks with Node-Cron
